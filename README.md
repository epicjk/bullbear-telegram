# Bull vs Bear - Telegram Mini App

BTC 가격 방향 예측 게임. 텔레그램 미니앱으로 플레이!

## 🎮 게임 방법

1. BTC 가격이 오를지(🐂 BULL) 내릴지(🐻 BEAR) 예측
2. 30초 안에 베팅
3. 60초 후 결과 확인
4. 맞추면 1.95배 수익!

## 🚀 실행

```bash
npm install
npm run dev
```

## 📱 텔레그램 연동

1. BotFather에서 봇 생성
2. `/newapp` 으로 WebApp 등록
3. URL: 배포된 Vercel URL 입력

## 🛠 기술 스택

- Next.js 14
- TypeScript
- Tailwind CSS
- @tma.js/sdk (Telegram Mini App SDK)
